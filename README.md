# Binance Futures Testnet Trading Bot

A structured Python trading bot for Binance Futures Testnet.

## Features

- Place MARKET orders
- Place LIMIT orders
- Logging with error handling
- CLI-based execution
- Clean modular structure

## Tech Stack

- Python
- python-binance
- dotenv
- Logging module
